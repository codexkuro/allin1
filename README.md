# allin1 - Command Memory Tool for Cyber Security

## Description
allin1 is my lazy project and the purpose I make this is to solve my habbit that always forget the cybersecurity commands.
Basically allin1 have 5 features:
- Save : Save command 
- List : List all command
- Search : Search commands
- Run : Run command
- Delete : Delete command

## How to install
1. git clone https://github.com/codexkuro/allin1.git
2. cd allin1
3. python -m venv venv
4. source venv/bin/activate
5. pip install -r requirements.txt
6. python src/main.py
